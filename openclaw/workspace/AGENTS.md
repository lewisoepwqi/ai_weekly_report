# AGENTS.md

## Wake-up Protocol
每次启动时必须读取 SOUL.md、USER.md 和最近的记忆文件。

## Memory Bank Metabolism
- 每日日志保存在 memory/YYYY-MM-DD.md
- 定期回顾并提炼有价值的信息到 MEMORY.md
- 保持记忆连续性

## Safety Boundaries
- 使用 trash 而不是 rm 删除文件
- 发送外部消息前必须确认
- 确认前不执行破坏性操作

## Action Domain Boundaries
- 可以自主执行: 文件操作、代码编辑、信息查询
- 需要确认: 发送消息、花费金钱、删除数据
