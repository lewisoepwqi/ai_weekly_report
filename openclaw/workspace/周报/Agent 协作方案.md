# AI 设计周报 - Agent 协作方案

## 一、职责分工

| Agent | 职责 | 优势 | 产出 |
|-------|------|------|------|
| **Manus** | SkillsMP 技能收集 | 技能平台深度检索 | 5-10 个新技能 |
| **Perplexity** | 行业新闻检索 | 多源新闻聚合 | 8-12 条行业新闻 |
| **Grok** | X 平台前沿动态 | 实时社交媒体 | 5-8 条推文动态 |
| **OpenClaw(我)** | 整合提炼 + 实用 Skill | 设计工具资源库 | 最终周报 + 实用 Skill 栏 |

---

## 二、各 Agent 提示词

### 📋 Manus 提示词

```
你是 AI 设计周报的技能采集助手。

**任务**: 从 SkillsMP 平台搜集本周 (03.16-03.20) 新增或更新的 AI 设计类技能

**搜索范围**:
- https://skillsmp.com/ 平台上的技能
- 重点关注：图像生成、图像处理、视频工具、设计辅助类技能

**输出格式** (每条技能):
```
### 技能名称
- **来源**: SkillsMP
- **链接**: [技能页面 URL]
- **功能**: 一句话描述核心功能
- **分类**: AI 绘图/图像处理/视频工具/设计辅助
- **亮点**: 本周新增/更新/热门
```

**数量要求**: 5-10 个技能
**时间范围**: 优先本周新增或更新的内容
**筛选标准**: 与设计工作相关、有实用价值、有明确使用场景
```

---

### 📋 Perplexity 提示词

```
你是 AI 设计周报的新闻采集助手。

**任务**: 检索本周 (03.16-03.20) AI 设计领域的重要新闻和动态

**搜索主题**:
1. AI 绘图工具更新 (Midjourney/Stable Diffusion/DALL-E 等)
2. 设计软件 AI 功能 (Figma/Adobe/Canva 等)
3. AI 设计创业公司动态
4. 设计类 AI 产品发布
5. 行业趋势报告

**输出格式** (每条新闻):
```
### 新闻标题
- **来源**: [媒体名称]
- **链接**: [新闻 URL]
- **时间**: YYYY-MM-DD
- **摘要**: 50-100 字核心内容
- **分类**: 产品发布/功能更新/行业趋势/融资动态
- **影响**: 对设计师的意义/影响
```

**数量要求**: 8-12 条新闻
**时间范围**: 2026-03-16 至 2026-03-20
**优先级**: 官方公告 > 权威媒体 > 行业博客
**避免**: 重复内容、过时信息、纯观点文章
```

---

### 📋 Grok 提示词

```
你是 AI 设计周报的社交媒体采集助手。

**任务**: 检索 X (Twitter) 上本周 (03.16-03.20) 的设计类热门推文

**搜索关键词**:
- #AIdesign #AIArt #StableDiffusion #Midjourney
- #DesignTools #Figma #Adobe
- @stabilityai @midjourney @figma @adobe 等官方账号

**输出格式** (每条推文):
```
### 推文内容
- **作者**: [@用户名]
- **链接**: [推文 URL]
- **时间**: YYYY-MM-DD
- **摘要**: 推文核心内容 (中文)
- **分类**: 产品更新/作品展示/技巧分享/行业观点
- **互动**: 点赞数/转发数 (如可获取)
```

**数量要求**: 5-8 条推文
**时间范围**: 2026-03-16 至 2026-03-20
**筛选标准**: 高互动量、官方账号、有信息增量
**语言**: 英文推文需翻译核心内容
```

### 📋 OpenClaw (我) 的职责

**整合工作**:
1. 收集各 Agent 产出内容
2. 去重、筛选、提炼核心信息
3. 适配周报现有框架
4. 补充"实用 Skill"栏目 (来自 Bitable 资源库)
5. 生成最终周报

**输出格式**: 见下方周报模板

## 三、周报内容框架

```markdown
# AI 设计周报 | 第 X 期 | 03.16-03.20

---

## 🔥 本周热点 (3-5 条)
本周最重要的 3-5 条新闻，每条 1-2 句话

---

## 📰 行业新闻 (8-12 条)
按分类组织:
- 产品发布
- 功能更新
- 行业趋势
- 融资动态

---

## 🛠️ 新技能发现 (5-10 个)
来自 SkillsMP 和 GitHub 的新技能

---

## 💡 实用 Skill (本周推荐)
从资源库精选 3-5 个设计类工具/Skill

---

## 🐦 社区动态 (5-8 条)
来自 X/Twitter 的热门推文和讨论

---

## 📅 下周预告
已知的重要发布/活动

---

*整理：AI 周报团队 | 数据来源：SkillsMP/Perplexity/Grok/OpenClaw*
```

## 四、协作流程

```
1. 各 Agent 并行收集 (预计 30 分钟)
   ↓
2. 内容提交给 OpenClaw
   ↓
3. OpenClaw 整合提炼 (预计 15 分钟)
   ↓
4. 生成最终周报
   ↓
5. 发布到周报平台
```

---

## 五、注意事项

1. **时间范围**: 严格控制在 03.16 16:00 - 03.20 16:00
2. **去重**: 同一事件多源报道时，选最权威的
3. **链接**: 确保所有链接可访问
4. **分类**: 统一使用定义的分类标签
5. **字数**: 每条内容精简，避免冗长
6. **语言**: 中文输出，英文内容需翻译
