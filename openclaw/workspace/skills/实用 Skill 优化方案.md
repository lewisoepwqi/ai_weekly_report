# 实用 Skill 模块优化方案

## 一、模块定位

**名称**: 实用 Skill（原"开源精选"升级）  
**目标**: 专注搜集辅助**设计工作**的 Skill、插件、SDK 资源，服务设计师和创意工作者  
**原则**: 精选、实用、易上手、设计向  
**场景**: 周报"开源精选"板块的内容源升级

---

## 二、内容源优化

### 2.1 主要来源渠道

| 渠道 | 优先级 | 更新频率 | 说明 |
|------|--------|----------|------|
| ClawHub | ⭐⭐⭐ | 每日 | https://clawhub.com 官方技能市场 |
| GitHub 官方仓库 | ⭐⭐⭐ | 每周 | openclaw 组织及示例仓库 |
| GitHub 搜索 | ⭐⭐ | 每周 | 包含 SKILL.md 的公开仓库 |
| 社区推荐 | ⭐⭐ | 随时 | Discord/飞书群用户投稿 |
| 第三方平台 | ⭐ | 每月 | 类似 Agent 技能平台 |

### 2.2 GitHub 搜索策略

```bash

```

### 2.3 设计类资源重点渠道

| 渠道 | 类型 | 说明 |
|------|------|------|
| GitHub Topics | design-tools, image-processing | 设计工具标签下的仓库 |
| Product Hunt | Design Tools 分类 | 每日新设计工具发布 |
| 设计师社区 | 站酷、UI 中国、Dribbble | 设计师分享的工具流 |
| AI 工具导航 | futurepedia.io 等 | AI 设计工具聚合站 |

### 2.4 筛选标准

**通用标准:**
- ✅ 包含完整的 SKILL.md 文件
- ✅ 有清晰的功能说明和使用示例
- ✅ 代码结构规范，有注释
- ✅ 近期有更新维护（6 个月内）
- ✅ 无安全风险提示
- ⭐ 优先选择 Star 数>10 的仓库

**设计类专项标准:**
- ✅ 解决实际设计工作痛点
- ✅ 支持常见设计软件/格式 (PSD/AI/Figma 等)
- ✅ 输出质量可控（图片/视频清晰度等）
- ✅ 有设计工作流整合能力
- ⭐ 优先选择设计师社区推荐的工具

## 七、风险与注意事项

1. **版权风险**: 确保收录的技能为开源许可，标注原作者
2. **安全风险**: 不收录涉及敏感操作或未经审计的代码
3. **维护风险**: 定期检查链接有效性，清理失效资源
4. **质量风险**: 建立审核机制，避免低质量技能流入
